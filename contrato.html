1. Objeto del servicio

El proveedor realizará actividades de monitoreo orientadas a supervisar el estado operativo y disponibilidad del sitio web autorizado por el cliente.

El servicio podrá incluir:

  • Supervisión periódica del estado del sitio web.
  • Verificación de disponibilidad (uptime).
  • Detección de errores de acceso o funcionamiento visibles externamente.
  • Notificaciones al cliente cuando se detecten incidencias.
  • Entrega de reportes o registros del monitoreo (cuando aplique).

3. Limitaciones del servicio

El monitoreo tiene carácter preventivo e informativo.

  • No se garantiza disponibilidad continua del sitio web.
  • No se garantiza prevención de caídas o incidentes.
  • No se incluyen correcciones técnicas salvo acuerdo adicional.
  • El proveedor no asume responsabilidad por fallos del hosting, terceros o infraestructura del cliente.

Las alertas y reportes tienen como objetivo facilitar una respuesta oportuna.

4. Autorización

El cliente declara ser propietario del sitio web o contar con autorización suficiente para solicitar el servicio de monitoreo.

El cliente autoriza expresamente al proveedor a realizar consultas automatizadas necesarias para prestar el servicio.

5. Confidencialidad

Toda información obtenida durante la prestación del servicio será tratada de manera confidencial y utilizada únicamente para fines relacionados con el monitoreo contratado.

6. Uso ético y buenas prácticas

El proveedor realizará el monitoreo procurando minimizar el impacto sobre el sitio web.

No se efectuarán pruebas destructivas ni actividades que alteren el funcionamiento normal del sistema.
