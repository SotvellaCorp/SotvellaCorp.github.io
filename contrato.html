Contrato de Servicio

Este acuerdo establece las condiciones del servicio de auditoría de seguridad web entre el proveedor y el cliente.

1. Objeto del servicio

El proveedor realizará pruebas básicas de seguridad orientadas a identificar posibles vulnerabilidades en el sitio web autorizado.

2. Alcance

  • Escaneo automatizado
  • Pruebas manuales básicas
  • Revisión de configuraciones comunes
  • Entrega de informe técnico

3. Autorización

El cliente declara ser propietario del sitio web o contar con autorización legal para solicitar esta auditoría.

4. Confidencialidad

Toda la información obtenida durante la auditoría será tratada de manera confidencial.

5. Limitación

El proveedor no garantiza la detección total de vulnerabilidades existentes ni será responsable por daños derivados del uso indebido del sistema.

6. Uso ético

Las pruebas se realizarán únicamente con fines preventivos y dentro del alcance autorizado.

7. Aceptación

Ambas partes aceptan los términos descritos en este acuerdo.
