About me

I am an Electronic Engineer graduated from the Universidad Pedagógica y Tecnológica de Colombia (UPTC), with practical training and strong interest in web security and vulnerability analysis.

I have developed hands-on experience through offensive security laboratories focused on web applications, including practical exercises and learning environments from PortSwigger Web Security Academy, where I have worked on identifying common security issues such as:

  • Authentication and access control flaws
  • Insecure configuration issues
  • Web logic vulnerabilities
  • Session and authentication analysis
  • Basic testing of forms and web parameters

I also have experience performing analysis and technical documentation of potential vulnerabilities on Bug Bounty platforms, focusing on web application review and the creation of clear and organized reports.

In addition, I maintain a personal practice laboratory based on Kali Linux virtual machines, where I use specialized tools for web security analysis and continuous learning, including:

  • Burp Suite
  • OWASP ZAP
  • Nikto
  • Nmap
  • Virtualized testing environments

My focus is oriented toward helping identify basic web security risks, improving configurations, and contributing to safer web environments for users and businesses.
