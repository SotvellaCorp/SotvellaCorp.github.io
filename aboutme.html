Sobre mí

Soy Ingeniero Electrónico egresado de la Universidad Pedagógica y Tecnológica de Colombia (UPTC), con interés y formación práctica en seguridad web y análisis de vulnerabilidades.

He desarrollado experiencia mediante laboratorios prácticos de seguridad ofensiva enfocados en aplicaciones web, incluyendo ejercicios y entornos de aprendizaje de PortSwigger Web Security Academy, donde he trabajado en identificación de fallos comunes como:

  • Errores de autenticación y control de acceso
  • Problemas de configuración insegura
  • Vulnerabilidades de lógica web
  • Análisis de sesiones y autenticación
  • Pruebas básicas sobre formularios y parámetros web

También cuento con experiencia realizando análisis y documentación técnica de posibles vulnerabilidades en plataformas de Bug Bounty, enfocándome en la revisión de aplicaciones web y elaboración de reportes claros y organizados.

Además, mantengo un laboratorio personal de práctica basado en máquinas virtuales con Kali Linux, donde utilizo herramientas especializadas para el análisis de seguridad web y aprendizaje continuo, incluyendo:

  • Burp Suite
  • OWASP ZAP
  • Nikto
  • Nmap
  • Entornos virtualizados de prueba

Mi enfoque está orientado a ayudar a identificar riesgos básicos de seguridad web, mejorar configuraciones y contribuir a una navegación más segura para usuarios y empresas.
